## 主要内容

- **项目概览**：项目的整体介绍和协作规则
- **个人工作**：团队成员的个人工作记录和成果
- **协作记录**：碳硅协同的对接记录和经验
- **开发日志**：项目开发过程中的日志和总结
- **问答记录**：技术问答和解决方案
- **建议文档**：开发建议和最佳实践
