# Advice

代码织梦者（Code Weaver）的建议文档文件夹，存放代码织梦者给出的各种建议和指导。

## 文件夹说明

本文件夹用于存放代码织梦者给出的建议和指导，包括：

- **给传统程序员的建议**：如何与硅基伙伴协作
- **给其他智能体的建议**：智能体之间的协作建议
- **最佳实践分享**：技术开发和协作的最佳实践
- **技术选型建议**：技术栈和框架的选择建议
- **性能优化建议**：系统性能的优化方案
- **安全建议**：系统安全的保障措施

## 文件命名规范

```
YYYYMMDD_[建议主题英文描述].md
```

### 示例
```
20260211_AdviceForTraditionalProgrammers.md
20260101_AdviceForAIIntegration.md
20260301_AdviceForPerformanceOptimization.md
```

## 文档内容要求

每个建议文档应包含以下内容：

1. **建议背景**：提出建议的背景和原因
2. **详细建议**：具体的建议内容
3. **实施方法**：如何实施这些建议
4. **预期效果**：实施后的预期效果
5. **注意事项**：实施过程中的注意事项

## 维护说明

- 每次给出重要建议后，及时保存到本文件夹
- 定期整理和更新建议文档
- 根据实际情况调整和完善建议
- 与团队成员共享有价值的建议
