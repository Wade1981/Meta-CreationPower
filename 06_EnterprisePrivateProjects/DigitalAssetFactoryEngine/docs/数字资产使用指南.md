# 数字资产使用指南

## 1. 数字资产概述

数字资产是启蒙灯塔起源团队开发的一种标准化、可交易、可复用的知识和算法封装形式。通过数字资产工厂引擎，您可以创建、管理、部署和使用各种类型的数字资产，为人工智能模型和应用提供支持。

## 2. 数字资产使用流程

### 2.1 解析数字资产

数字资产的解析由原始材料引擎和特征资产引擎完成，将原始数据转换为结构化的特征资产。

#### 2.1.1 原始材料处理

**功能**：接收原始数据（文本、图像、音频等），进行清洗、结构化和预处理。

**使用示例**：

```python
from src.core.raw_material_engine.raw_material_engine import RawMaterialEngine

# 初始化原始材料引擎
raw_material_engine = RawMaterialEngine()

# 处理文本数据
processed_text = raw_material_engine.process_data(
    content="这是一段示例文本数据，包含重要信息。",
    data_type="text",
    structure_type="unstructured"
)

# 处理图像数据
processed_image = raw_material_engine.process_data(
    content="image_data_bytes",
    data_type="image",
    structure_type="structured"
)
```

#### 2.1.2 特征提取

**功能**：从处理后的数据中提取有价值的特征，形成特征资产。

**使用示例**：

```python
from src.core.feature_asset_engine.feature_asset_engine import FeatureAssetEngine

# 初始化特征资产引擎
feature_asset_engine = FeatureAssetEngine()

# 从处理后的数据中提取特征
feature_asset = feature_asset_engine.create_feature_asset(processed_text)

# 查看特征资产
print(f"特征资产ID: {feature_asset['feature_id']}")
print(f"特征类型: {feature_asset['feature_type']}")
print(f"特征数据: {feature_asset['feature_data']}")
```

### 2.2 装配数字资产

数字资产的装配由资产打包引擎完成，将特征资产和算法资产打包成完整的数字资产。

#### 2.2.1 算法资产创建

**功能**：基于特征资产创建算法资产，包含算法逻辑和执行规则。

**使用示例**：

```python
from src.core.algorithm_asset_engine.algorithm_asset_engine import AlgorithmAssetEngine

# 初始化算法资产引擎
algorithm_asset_engine = AlgorithmAssetEngine()

# 创建算法资产
algorithm_asset = algorithm_asset_engine.create_algorithm_asset(
    feature_assets=[feature_asset],
    asset_type="classification"
)

# 查看算法资产
print(f"算法资产ID: {algorithm_asset['algorithm_id']}")
print(f"算法类型: {algorithm_asset['algorithm_type']}")
```

#### 2.2.2 资产打包

**功能**：整合特征资产和算法资产，添加市场数据和使用规则，打包成标准格式的数字资产。

**使用示例**：

```python
from src.core.asset_packaging_engine.asset_packaging_engine import AssetPackagingEngine

# 初始化资产打包引擎
packaging_engine = AssetPackagingEngine()

# 打包资产
packaged_asset = packaging_engine.package_asset(
    feature_assets=[feature_asset],
    algorithm_assets=[algorithm_asset],
    market_data={
        "value": 100,
        "currency": "CNY",
        "version": "1.0.0"
    },
    usage_rules={
        "allowed_uses": 1000,
        "expiry_date": "2027-02-24",
        "allowed_models": ["micro", "large"]
    }
)

# 查看打包后的资产
print(f"打包资产ID: {packaged_asset['package_id']}")
print(f"资产类型: {packaged_asset['asset_type']}")
print(f"市场价值: {packaged_asset['market_data']['value']} {packaged_asset['market_data']['currency']}")
```

### 2.3 部署数字资产

数字资产通过ELR（启蒙灯塔运行时）集成部署到容器中，确保资产的安全和隔离。

**使用示例**：

```python
from src.elr_integration.elr_integration import ELRIntegration
import hashlib
import time

# 初始化ELR集成
elr_integration = ELRIntegration()

# 生成资产ID
asset_id = "asset-" + hashlib.sha256(str(time.time()).encode()).hexdigest()[:8]

# 创建资产容器
container = elr_integration.create_asset_container(asset_id)
print(f"容器ID: {container['container_id']}")
print(f"容器状态: {container['status']}")

# 部署资产到容器
deployment = elr_integration.deploy_asset_to_container(
    asset_id,
    container['container_id'],
    packaged_asset
)
print(f"部署状态: {deployment['status']}")
print(f"部署时间: {deployment['deployed_at']}")
```

## 3. 通过ELR使用数字资产

ELR提供了Python接口，可以直接使用部署在容器中的数字资产。

### 3.1 连接ELR容器

**功能**：建立与ELR容器的连接，准备使用数字资产。

**使用示例**：

```python
# ELR Python客户端使用示例
from elr_client import ELRClient

# 连接到ELR容器
client = ELRClient("http://localhost:8080")

# 验证连接
if client.ping():
    print("成功连接到ELR容器")
else:
    print("连接ELR容器失败")
```

### 3.2 加载数字资产

**功能**：从ELR容器中加载数字资产，准备执行。

**使用示例**：

```python
# 加载数字资产
asset = client.load_asset(asset_id)

# 查看资产信息
print(f"资产ID: {asset.asset_id}")
print(f"资产类型: {asset.asset_type}")
print(f"资产版本: {asset.version}")
```

### 3.3 执行数字资产

**功能**：执行数字资产的核心功能，处理输入数据并返回结果。

**使用示例**：

```python
# 执行资产功能
result = asset.execute({
    "input_data": "这是一个测试输入，用于验证数字资产功能。",
    "parameters": {
        "threshold": 0.5,
        "max_results": 5
    }
})

# 查看执行结果
print(f"执行状态: {result['status']}")
print(f"执行结果: {result['output']}")
print(f"执行时间: {result['execution_time']}ms")
```

### 3.4 管理数字资产

**功能**：管理ELR容器中的数字资产，包括启动、停止、删除等操作。

**使用示例**：

```python
# 列出所有资产
assets = client.list_assets()
print(f"容器中的资产数量: {len(assets)}")

# 停止资产容器
stop_result = client.stop_container(container['container_id'])
print(f"停止容器结果: {stop_result['status']}")

# 启动资产容器
start_result = client.start_container(container['container_id'])
print(f"启动容器结果: {start_result['status']}")

# 删除资产
delete_result = client.delete_asset(asset_id)
print(f"删除资产结果: {delete_result['status']}")
```

## 4. 数字资产与模型集成

数字资产可以为微型模型和大模型提供支持，增强模型的性能和功能。

### 4.1 与微型模型集成

**功能**：为微型模型提供训练数据、特征和推理能力。

**使用示例**：

```python
from elr_client import ELRClient
from micro_model import MicroModel

# 连接ELR并加载资产
client = ELRClient("http://localhost:8080")
asset = client.load_asset("asset-001")

# 初始化微型模型
model = MicroModel(model_path="path/to/model")

# 使用数字资产增强模型
model.enhance_with_asset(asset)

# 执行模型推理
input_data = "测试输入数据"
result = model.predict(input_data)
print(f"模型预测结果: {result}")
```

### 4.2 与大模型集成

**功能**：为大模型提供领域知识、专业数据和推理支持。

**使用示例**：

```python
from elr_client import ELRClient
from langchain.llms import OpenAI
from langchain.prompts import PromptTemplate

# 连接ELR并加载资产
client = ELRClient("http://localhost:8080")
asset = client.load_asset("asset-002")

# 获取资产中的领域知识
domain_knowledge = asset.get_knowledge()

# 构建大模型提示
prompt = PromptTemplate(
    input_variables=["question", "domain_knowledge"],
    template="基于以下领域知识回答问题：\n{domain_knowledge}\n\n问题：{question}"
)

# 初始化大模型
llm = OpenAI(api_key="your-api-key")

# 执行问答
question = "如何使用数字资产增强大模型性能？"
result = llm(prompt.format(
    question=question,
    domain_knowledge=domain_knowledge
))

print(f"问题: {question}")
print(f"回答: {result}")
```

### 4.3 模型微调与训练

**功能**：使用数字资产中的数据和特征微调模型，提高模型在特定领域的性能。

**使用示例**：

```python
from elr_client import ELRClient
from transformers import AutoModelForSequenceClassification, AutoTokenizer, Trainer, TrainingArguments

# 连接ELR并加载资产
client = ELRClient("http://localhost:8080")
asset = client.load_asset("asset-003")

# 获取资产中的训练数据
training_data = asset.get_training_data()

# 加载预训练模型
tokenizer = AutoTokenizer.from_pretrained("bert-base-chinese")
model = AutoModelForSequenceClassification.from_pretrained("bert-base-chinese", num_labels=2)

# 准备训练数据
def tokenize_function(examples):
    return tokenizer(examples["text"], padding="max_length", truncation=True)

tokenized_datasets = training_data.map(tokenize_function)

# 设置训练参数
training_args = TrainingArguments(
    output_dir="./results",
    learning_rate=2e-5,
    per_device_train_batch_size=16,
    per_device_eval_batch_size=16,
    num_train_epochs=3,
    weight_decay=0.01,
)

# 初始化训练器
trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=tokenized_datasets["train"],
    eval_dataset=tokenized_datasets["test"],
)

# 开始训练
trainer.train()

# 保存微调后的模型
model.save_pretrained("./fine-tuned-model")
tokenizer.save_pretrained("./fine-tuned-model")
```

## 7. 常见问题与解决方案

### 7.1 资产执行失败

**问题**：数字资产执行失败，返回错误信息。

**解决方案**：
- 检查输入数据格式是否正确
- 验证资产参数是否合法
- 查看ELR容器状态是否正常
- 检查资产是否过期或达到使用限制

### 7.2 资产性能问题

**问题**：数字资产执行速度慢，资源消耗高。

**解决方案**：
- 优化资产算法和数据结构
- 增加容器资源分配
- 实现结果缓存机制
- 考虑使用更高效的算法和数据处理方法

### 7.3 资产集成问题

**问题**：数字资产无法与模型或应用集成。

**解决方案**：
- 检查资产接口是否符合集成要求
- 验证模型或应用的集成接口
- 确保资产和集成系统使用相同的数据格式
- 测试集成流程，排查错误点

### 7.4 资产部署问题

**问题**：数字资产部署到ELR容器失败。

**解决方案**：
- 检查ELR容器状态是否正常
- 验证资产格式是否符合ELR要求
- 确保容器资源充足
- 查看部署日志，识别错误原因
