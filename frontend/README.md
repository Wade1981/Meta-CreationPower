# 数字资产工厂引擎前端界面

## 项目简介

本项目是数字资产工厂引擎的前端界面，基于HTML、CSS和JavaScript开发，使用Bootstrap 5和Chart.js构建现代化界面。前端界面提供了直观的用户操作界面，支持数字资产的管理、系统状态监控和工作流执行。

## 功能特性

### 1. 仪表盘
- 展示系统状态和关键指标
- 总资产数、活跃进程、总交易数
- 资产分布图表

### 2. 资产管理
- 资产列表展示
- 创建新资产
- 编辑和删除资产
- 资产搜索功能

### 3. 系统状态
- 模块运行状态监控
- 容器状态展示

### 4. 工作流管理
- 工作流执行
- 执行历史记录

## 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **UI框架**：Bootstrap 5
- **图表库**：Chart.js
- **数据处理**：模拟API数据

## 快速开始

### 1. 打开前端界面

直接在浏览器中打开 `index.html` 文件即可访问前端界面。

### 2. 功能使用

#### 仪表盘
- 查看系统概览和关键指标
- 查看资产分布图表

#### 资产管理
- 点击"创建资产"按钮创建新资产
- 填写资产信息并提交
- 查看、编辑或删除现有资产
- 使用搜索框搜索资产

#### 系统状态
- 查看各模块的运行状态
- 查看容器状态

#### 工作流
- 点击"执行工作流"按钮
- 输入工作流ID和数据
- 查看工作流执行历史

## 注意事项

1. 本前端界面使用模拟数据，无需后端服务器即可展示所有功能
2. 若要使用真实后端API，请修改 `app.js` 中的 `apiRequest` 函数，取消注释实际API调用代码
3. 后端服务器需要Python和Flask环境
